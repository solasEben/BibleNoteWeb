## üöÄ Getting Started

### Prerequisites

- Node.js (LTS version)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/solasEben/BibleNoteWeb.git
   ```
2. Navigate to the project directory:
   ```bash
   cd BibleNoteWeb
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
